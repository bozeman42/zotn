# Zombies of the North

## Description
Zombies of the North is a multiplayer, hunters vs zombies game to be played at conventions. This application manages the players and all game assets. At this time I am not granting any license for this software. Copyright Â© 2018 Aaron Kvarnlov-Leverty

## Instructions
1. run `yarn install` in the root directory
2. In another terminal window, navigate to the server directory and run `yarn start`
3. Run `yarn start` in the root directory
4. Navigate to `localhost:5000`
